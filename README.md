# AI语音绘图工具

一个纯语音控制的 Web 绘图原型。用户点击麦克风开始持续说中文指令，再点一次结束录音并由浏览器语音识别转写，然后由本地规则解析器生成结构化命令，最后交给 Fabric.js 画布执行。

## 功能

- 点击麦克风开始持续监听，再点一次识别并执行。
- 实时显示识别文本、语音状态和执行反馈。
- 支持矩形、正方形、圆形、三角形、直线、文字和简单语义组合图。
- 支持颜色、尺寸、位置、相对位置、移动、删除、清空、撤销、重做和导出 PNG。
- 支持模糊指令确认，例如“画一个红色的”后可回答“第一个”或“第二个”。
- 预留 `src/ai/aiCommandAdapter.ts`，后续可接入 LLM 或生图服务。

## 常用语音指令

- “画一个红色圆形”
- “画一个蓝色正方形，边长200，放在左上角”
- “在圆形右边画一个三角形”
- “添加文字你好”
- “把这个矩形变成蓝色”
- “向左移动一点”
- “删除这个圆形”
- “全部清空”
- “撤销刚才的操作”
- “重做”
- “导出”

## 开发

```bash
npm install
npm run dev
```

## 验证

```bash
npm run test:run
npm run build
```

## 浏览器要求

语音识别使用 Web Speech API，建议使用最新版 Chrome 或 Edge。若浏览器不支持 `SpeechRecognition` 或 `webkitSpeechRecognition`，应用会显示不可用提示，画布和辅助按钮仍可工作。
